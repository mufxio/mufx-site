Draft 1: The Organism

Premise

The system is not a tool that analyzes the market. The system IS an organism
that lives inside the market's ecology. It breathes price, metabolizes
uncertainty, and excretes predictions. The trader's 93% sight-reading edge
is the organism's consciousness — this architecture builds the autonomic
nervous system underneath it.

Everything runs on a single Mac Mini. No cloud. No budget. The constraint
is the feature: a lean organism survives where bloated ones starve.

I. SENSORY CORTEX — Perceive

The organism's interface with raw market data. It never passes raw data
upward. Each layer processes a different timescale and passes only
summaries and errors.

The organism does not trade. The TRADER trades. The organism prepares
the trader's body for action. It outputs signals that enhance the
trader's sight-reading, not replace it.

IV. IMMUNE SYSTEM — Protect

The organism must defend itself against two threats: regime change it
hasn't detected, and its own overfitting.

|                                                               |

|  accuracy I'm getting?"                                       |
|                                                               |
|  If complexity rises without accuracy gain --> PRUNE          |
|  If accuracy drops without complexity change --> REGIME SHIFT |
|                                                               |
|  This IS Occam's razor from physics. The simplest model       |
|  that still works wins. [18]                                  |
|                                                               |
|  COMPOUND [18 x 11]: free energy computed FROM prediction     |
|  error. Rising free energy = the organism's model is          |
|  drifting from reality. This triggers the immune response.    |
+---------------------------------------------------------------+
    |
    v

VI. SKELETAL STRUCTURE — Support

The deep grammar that everything else hangs on. This is not a subsystem
— it is the architecture's spine.

+---------------------------------------------------------------+
|  THE TREE [25, 26, 29, 31, 32]                                |
|                                                               |
|  The market looks like a chain (tick, tick, tick).             |
|  But underneath the chain is a TREE. [25]                     |
|  Seeing the tree behind the chain IS expertise. [25]          |
|                                                               |
|  Markov = degenerate tree with branching factor 1. [25]       |
|  The tree is richer. Same surface event, different             |
|  structural meaning depending on tree position. [26, 31]      |
|                                                               |
|              [ROOT: current regime]                            |
|              /         |          \                            |
|         [trending]  [ranging]  [transitioning]                |
|          /    \       / \          |                           |
|     [impulse] [     [   ]    [breakout]                       |
|      /    \   correct]                / \                      |
|  [strong][weak]               [real]  [false]                 |
|     |      |                    |        |                    |
|   candle  candle             candle   candle                  |
|                                                               |
|  Same candle. Different tree position.                        |
|  Different meaning. [26, 31]                                  |
|                                                               |
|  PROLONGATION [26]: one structural node (e.g., "trending")    |
|  governs many surface events. You don't trade candles.        |
|  You trade the structural node that GOVERNS the candles.      |
|                                                               |
|  CONTEXT-FREE RECURSION [3]: the production rules call        |
|  themselves. A consolidation inside a consolidation inside    |
|  a trend uses the SAME rule at different parse depths.        |
|  Same terminal symbol, different parse tree position,          |
|  different meaning. [3]                                       |
|                                                               |
|  Production rules:                                             |
|    Regime  --> Trend | Range | Transition                     |
|    Trend   --> Impulse Correction Trend | Impulse             |
|    Range   --> Test_High Test_Low Range | Breakout            |
|    Impulse --> Strong | Weak                                  |
|    (these rules call themselves = recursion) [3]              |
|                                                               |
|  BEHAVIOR TREE [31]: position in tree determines meaning.     |
|  A "strong impulse" under "Trend > Impulse" = continuation.  |
|  A "strong impulse" under "Range > Breakout" = initiation.   |
|  Same action, different parent, different meaning. [31]       |
|                                                               |
|  MARKOV BLANKET [29]: the tree grows INSIDE a Markov          |
|  blanket. The boundary (what the organism exchanges with      |

|  The organism is an autonomous entity: tree-structured        |
|  processing wrapped in a Markov-membraned skin. [29]          |
|                                                               |
|  NEXT-TOKEN PREDICTION [32]: if the data is tree-structured   |

|  the next tick, over and over, until the tree emerges.        |
|                                                               |
|  COMPOUND [32 x 3 x 25]: next-token prediction [32]          |
|  discovers context-free recursive structure [3] that          |

Mechanic map:
- [25] Tree contains chain: Markov is degenerate tree

VII. STIGMERGIC MEMBRANE — The Boundary

The organism does not observe the market from outside. It lives IN the
market. Its observations modify the environment, and the modified
environment feeds back.

+---------------------------------------------------------------+
|  STIGMERGIC INTERFACE [7]                                     |
|                                                               |
|  Price IS the pheromone trail. [7]                             |
|                                                               |
|  Every market participant modifies the price.                  |
|  Every other participant reads the modified price.             |
|  No direct communication. Only indirect, through the           |
|  environment. [7]                                              |
|                                                               |
|  The organism's trades (when the trader acts) are              |
|  themselves pheromone deposits. The market reads them.         |
|  This creates a feedback loop:                                 |
|                                                               |
|  organism --> trade --> price change --> organism reads        |
|       ^                                        |              |
|       +----------------------------------------+              |
|                                                               |
|  PATH DEPENDENCE [27]: the pheromone trail has memory.         |
|  Four lock-in mechanisms:                                      |
|    1. Increasing returns: support/resistance levels            |
|       strengthen with each test                                |
|    2. Learning effects: participants learn the level,          |
|       making it more real                                      |
|    3. Coordination effects: everyone watches the same          |
|       levels, creating self-fulfilling prophecy                |
|    4. Adaptive expectations: expectations adapt to past        |
|       behavior, reinforcing it                                 |
|  The transition matrix DEPENDS ON THE PATH. [27]              |
|                                                               |
|  COMPOUND [7 x 27 x 12]: stigmergic traces [7] create        |
|  path-dependent [27] pattern completion [12]. The organism    |
|  encounters a partial price pattern and the hippocampus       |
|  completes it to the NEAREST STORED PATTERN [12]. But         |
|  which pattern is "nearest" depends on the path taken to      |
|  get here [27]. Pattern completion is biased toward            |
|  familiar patterns (inertia [12]). Error accumulates           |
|  until SNAP to new pattern. [12] This snap IS the             |
|  boundary event detected by the basal ganglia. [23]           |
+---------------------------------------------------------------+

Mechanic map:
- [7] Stigmergy: price as pheromone, indirect communication
- [27] Path dependence: four lock-in mechanisms
- [12] Pattern completion with inertia: hippocampal snap

OPTIC Equivalences — The Organism's Algebra [16]

Before any processing, raw market signals pass through five binary
transformations that strip away surface differences and expose pure
structural dynamics.

+---------------------------------------------------------------+
|  OPTIC TRANSFORMS [16]                                        |
|                                                               |
|  5 binary choices, each strips one dimension of difference:   |
|                                                               |
|  O: Original / Retrograde (forward / time-reversed)           |
|  P: Prime / Inverted (as-is / price-inverted)                 |
|  T: Transposed / Original (shifted in price level)            |
|  I: Inverted / Original (flipped about axis)                  |
|  C: Compressed / Expanded (time-scaled)                       |
|                                                               |
|  Apply all 5 = 2^5 = 32 equivalent views of the same         |
|  structural event. [16]                                        |
|                                                               |
|  A rising wedge and a falling wedge are OPTIC-equivalent.     |
|  A breakout at 1.0850 and a breakout at 1.1200 are            |
|  OPTIC-equivalent. Same structure, different surface. [16]    |
|                                                               |
|  COMPOUND [16 x 3 x 26]: OPTIC equivalences [16] reveal      |
|  that context-free recursion [3] produces the SAME terminal   |
|  symbol in OPTIC-equivalent positions. Prolongation [26]      |
|  becomes visible: one structural node governs many surface    |
|  events that LOOK different but are OPTIC-equivalent.         |
|  The tree's branching structure is invariant under OPTIC.     |
+---------------------------------------------------------------+

Mechanic map:
