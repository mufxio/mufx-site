Fast feedforward path. Perception → world model (single step) → actor. Like Kahneman's System 1 — pattern-matched, immediate.

Slow iterative planning. Actor proposes sequences, world model simulates outcomes, critic evaluates. Loops until optimal. Like System 2.

The Core Problem & Solution

Representation
Collapse
& how SIGReg solves it

If both encoders output constant vectors, loss goes to zero — but representations encode nothing. This is collapse. The field's entire history is the hunt for a principled solution.

Collapsed

Without regularization

Embeddings collapse to a single point. Loss → 0, information → 0.

SIGReg

With isotropic regularization

Embeddings spread to match isotropic Gaussian. Information maximized.

SIGReg: The Principled Fix

LeJEPA (2025) proves the isotropic Gaussian is the unique optimal distribution minimizing worst-case downstream task risk. SIGReg enforces this via the Cramér-Wold theorem.

Cramér-Wold Trick

A multivariate distribution is uniquely determined by all its 1D projections. SIGReg checks projections onto M random directions — O(N) complexity — and matches each to a standard normal.

What it eliminates

No EMA teacher. No stop-gradients. No whitening layers. No asymmetric architecture. Just one regularizer. Collapse is impossible by construction — not just empirically unlikely.

Collapse Prevention — Historical Arc

From negative pairs
to provable
guarantees.

Each generation found a simpler, more principled answer to the same question: how do you stop representations from degenerating?

🔄
2020
SimCLR / MoCo — Contrastive Learning
Prevent collapse by pushing apart negative pairs. Requires large batches (SimCLR) or memory banks (MoCo). Augmentation-dependent.
Problem: scales poorly; needs negatives; augmentation sensitivity
🎓
2020
BYOL — Bootstrap Your Own Latent
No negative pairs. Asymmetric architecture + EMA teacher. Mysteriously doesn't collapse. Works empirically — theory unclear at first.
Mystery: why doesn't it collapse? Nobody fully knew.
📊
2021
Barlow Twins (LeCun et al.)
Cross-correlation matrix between views should equal identity. Makes embedding dimensions maximally independent. Elegant — no negatives, no momentum, no asymmetry needed. 777 citations.
Innovation: decorrelation as the regularization objective
⚖️
2021
VICReg — Variance-Invariance-Covariance
Three explicit terms: keep variance above threshold, embeddings of augmented views match, embedding dims decorrelated. Most explicit collapse prevention. 285 citations.
Innovation: named what prevents collapse; no tricks needed
🖼️
2023
I-JEPA — First JEPA Implementation (CVPR)
Predict representations of masked image regions. EMA teacher provides stable targets. No augmentations. No pixel reconstruction. Semantic representations from masking alone. 254 citations.
Limitation: EMA is still a hack — no collapse guarantee
🎬
2024
V-JEPA — Video Extension (Meta AI)
Spatiotemporal masking across 64 frames. Mask consistent across time (prevents trivial same-frame copying). 1.5–6× more sample efficient than generative pre-training.
Key: mask repeats across time dimension — forces genuine temporal reasoning
🤖
2025
V-JEPA 2 — Understanding + Planning + Robot Control
1M+ hours internet video. Zero-shot robot arm control from 62 hours robot data. Action-conditioned planning via CEM in latent space. SotA on action anticipation.
Milestone: 62 hrs robot video → zero-shot Franka arm control across labs
✓
2025
LeJEPA + SIGReg — Provable Collapse Prevention
Theorem: isotropic Gaussian uniquely minimizes worst-case downstream risk. SIGReg enforces via Cramér-Wold projections. O(N) complexity. 1 hyperparameter. First JEPA with training-loss/downstream correlation.
✓ First mathematically provable guarantee — not empirical hope
🌍
2026
LeWorldModel (AMI Labs, LeCun) — 15M Params, Raw Pixels
First JEPA training stably end-to-end from raw pixels. Action-conditioned predictor = proper world model. 48× faster planning than foundation-model approaches. $1.03B seed round.
Democratization: world models on a single GPU

Core Objective

JEPA Prediction Loss

D(ŝ_y, s_y) = ||ŝ_y − s_y||²

Minimize MSE between predicted and actual representation. No pixel reconstruction. The entire insight in one line.

ŝ_y = predicted repr
s_y = target repr (EMA)
||·||² = L2 distance
Latent Variable

Predictor with Latent z

ŝ_y = g_φ(f_θ(x), z)

z handles multimodality — given a context, many futures are valid. z picks which one. Information capacity of z is deliberately bounded.

f_θ = context encoder
g_φ = predictor
z = latent (which future)
EMA Update

Target Encoder Update

θ̄ ← m·θ̄ + (1−m)·θ

Target encoder is a slowly-evolving copy of context encoder. Typical m ≈ 0.996–1.0. Prevents collapse empirically — but no formal guarantee (SIGReg fixes this).

θ̄ = target encoder params
θ = context encoder params
m = momentum ≈ 0.996
SIGReg — Collapse Prevention

LeJEPA Total Loss

L = L_predict + λ·L_SIGReg

L_SIGReg uses Epps-Pulley test on M random 1D projections of embeddings, matching to N(0,1). Cramér-Wold: matching all projections = matching the full distribution.

λ = single hyperparameter
M = random projections
N(0,I) = target distribution
Energy-Based View

EBM Framing

F_w(x,y) = min_z E_w(x,y,z)

JEPA is formally an energy-based model. Low energy = compatible (x,y) pair. No normalization required — more general than probabilistic models. Connects to statistical physics.

F_w = free energy
E_w = energy function
low E → compatible
LeWorldModel

Action-Conditioned World Model

ŝ_{t+1} = g_φ(s_t, a_t)

Given current state representation s_t and action a_t, predict next state representation. This IS a world model. Planning = find action sequence minimizing cost in latent space.

s_t = state repr at t
a_t = action
Plan via CEM in latent

The Key Bet JEPA Makes

Generation and understanding are separable. You can generate photorealistic video without understanding physics (diffusion models). And you can understand physics without generating video (JEPA). The path to understanding goes through compressed, abstract prediction — not through generative scaling.

Verdict: Partly right. Generative models work better than LeCun claimed (Sora). But understanding ≠ generation remains a valid philosophical point. The future is likely hybrid.
